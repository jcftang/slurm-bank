NAME

sbank-submit - sbank wrapper submit command

SYNOPSIS

sbank submit [OPTION]...

DESCRIPTION

Simple wrapper for submitting jobs, it takes a standard slurm batch script. The script gets checked to see if enough hours are available prior to job submission.

  • -s, --scriptname

    SBATCH script that you wish to submit.
