slurm-bank/ use case/ user

What the user Charles will want to know

sbank balance statement to see Charles' own balance. Apart from this one command Charles does not need to know much more about the slurm bank commands.

If Charles was a PI or Team leader in a project, he may want to see a more detailed balance sheet. He can do this by sbank balance statement -A
