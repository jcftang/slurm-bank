NAME

sbank - slurm bank top level command

SYNOPSIS

sbank subcommand [OPTION]...

DESCRIPTION

Top level command which wraps a bunch of subcommands for administering a slurmdbd. The set of commands also contains a bunch of commands for end users.

  • project

    List, create and delete projects in a defined way with parent=root, currently this is not customisable.

  • user

    List, create and users in a defined way.

  • deposit

    Add or remove hours from an account.

  • balance

    Show the balances of an account or a set of accounts.

  • time

    Time calculator.

  • cluster

    List current cluster or all clusters in slurmdbd, also capable of adding or removing clusters.
